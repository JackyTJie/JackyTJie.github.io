# JackyTJie.github.io

My personal portfolio website hosted at [tangjie.xyz](https://tangjie.xyz)

## Overview

This is my personal website and portfolio, showcasing my projects, skills, and contact information. The site is built with HTML, CSS, and JavaScript, and is hosted using GitHub Pages.

## Features

- **Responsive Design**: Works on all device sizes
- **Navigation Menu**: Easy navigation between sections
- **Skills Visualization**: Interactive pie charts showing skill levels
- **Project Showcase**: Links to my GitHub repositories
- **Gallery**: Display of personal images
- **Contact Section**: Multiple ways to get in touch
- **Weather Widget**: Real-time weather information for Shanghai
- **Background Animation**: Canvas-nest.js creates a dynamic background

## Sections

- **Home**: Welcome message and introduction
- **About**: Personal information and interests
- **Skills**: Visual representation of my technical abilities
- **Projects**: Showcase of my GitHub repositories
- **Gallery**: Collection of personal images
- **Contact**: Contact information and social links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [Canvas-Nest.js](http://cdn.bootcss.com/canvas-nest.js) for background animation
- [Vanilla-Tilt.js](https://github.com/micku7zu/vanilla-tilt.js) for 3D tilt effects
- [OpenWeatherMap API](https://openweathermap.org/api) for weather data

## Setup

This website is hosted on GitHub Pages. To run locally:

1. Clone the repository
2. Open `index.html` in your browser
